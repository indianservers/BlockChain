Module 1

What Is Blockchain?

Start with the mental model: a ledger that many participants share and protect together.

Blockchain in plain language

  • Blockchain is a digital ledger.
  • It stores data in blocks.
  • Blocks are connected in sequence.
  • Each block contains data, hash, previous hash, and timestamp.
  • Once recorded, data is difficult to change.
